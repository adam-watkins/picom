Documentation
RAIVEN's documentation was built using [mkdocs-material](https://squidfunk.github.io/mkdocs-material/). To install and start the development documentation server,
please first install mkdocs-material and then run the server as shown below.

<div class="termy">

```console
$ pip install mkdocs-material
---> 100%
$ cd docs/en
$ mkdocs serve
```

</div>
